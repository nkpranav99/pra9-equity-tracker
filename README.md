# Equity Trading Bot

A personal Indian equity trading assistant that integrates with Zerodha (Kite Connect), scrapes Chartink screeners for potential stock breakouts, runs secondary technical indicator validations, and communicates entirely via a Telegram Bot interface using Natural Language Processing (NLP).
